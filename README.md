- esxi-info.sh - sirve para listar las maquinas virtuales activas, ademas de mostrar el world ID, memoria asignada, memoria usada y consumo de CPU de cada una.
Forma de invocarlo: ./esxi-info.sh
- esxi-power_on.sh - sirve para encender una maquina virtual poniendo simplemente el nombre de esta en el propio comando.
Forma de invocarlo: ./esxi-power_on.sh "nombre maquina virtual"
- esxi-power_off.sh - sirve para apagar una maquina virtual poniendo simplemente el nombre de esta en el propio comando.
Forma de invocarlo: ./esxi-power_off.sh "nombre maquina virtual"
